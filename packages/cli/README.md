# @minions-social-accounts/cli

CLI for Minions Social-accounts — Account definitions, platforms, namespaces/purposes, and API keys

## Installation

```bash
npm install -g @minions-social-accounts/cli
```

## Usage

```bash
social-accounts --help
```
