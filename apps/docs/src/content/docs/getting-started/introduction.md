# Minions Social-accounts

Account definitions, platforms, namespaces/purposes, and API keys

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-social-accounts/sdk`) — Core library for Node.js
- **Python SDK** (`minions-social-accounts`) — Python bindings
- **CLI Tool** (`@minions-social-accounts/cli`) — Command-line interface
- **Web App** — Interactive playground at [social-accounts.minions.wtf](https://social-accounts.minions.wtf)
