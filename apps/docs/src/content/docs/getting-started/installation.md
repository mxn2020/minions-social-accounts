---
title: Installation
description: Install Minions Social-accounts packages
---

## TypeScript / Node.js

```bash
npm install @minions-social-accounts/sdk minions-sdk
```

## Python

```bash
pip install minions-social-accounts
```

## CLI

```bash
npm install -g @minions-social-accounts/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
